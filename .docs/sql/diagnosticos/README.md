## Scripts de diagnóstico

Consultas SQL destinadas a identificar problemas ou levantar estatísticas específicas.

### Conteúdo

- `check-duplicates.sql`: identifica registros duplicados em tabelas críticas.
- `check-invalids.sql`: encontra registros inválidos ou inconsistentes.
- `check-plan.sql`: (ex-`backend/check-plan.sql`) verifica planos e limites aplicados.

> Execute essas consultas primeiro em ambientes de homologação. Registre achados relevantes nos documentos de diagnóstico correspondentes (`.docs/diagnosticos/`).
