## Scripts de correção

Scripts destinados a aplicar ajustes pontuais em dados já existentes.

### Conteúdo

- `FIX-DUPLICATES.sql`: rotina abrangente para remover duplicidades.
- `QUICK-FIX-DUPLICATES-NOW.sql`: versão rápida de correção para incidentes urgentes.
- `remove-trigger.sql`: remove/ajusta triggers específicas conforme necessidade.
- `enable-campaigns.sql`: reabilita registros de campanhas conforme critérios definidos.

> Faça backup do banco antes de rodar qualquer script desta pasta. Documente a execução em `.docs/anexos/incidentes.md`.
